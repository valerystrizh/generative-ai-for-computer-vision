# Generative AI for Computer Vision

This repository contains course materials for **Deep Learning & Generative AI for Computer Vision**.
It covers both theoretical foundations and hands-on practice with modern deep learning models for visual data.

## Topics Covered
- Convolutional Neural Networks (CNNs)
- Vision Transformers (ViT, hybrid architectures)
- Representation learning with autoencoders
- Variational Autoencoders (VAEs)
- Generative Adversarial Networks (GANs)
- Diffusion models
- Self-supervised and masked image modeling
- Evaluation metrics for generative models

## Structure
- `lectures/` – Lecture slides and theoretical materials  
- `labs/` – Hands-on PyTorch notebooks  
- `projects/` – Practical assignments and mini-projects  
- `resources/` – Additional reading and references  

## Prerequisites
- Python programming
- Basic linear algebra and probability
- Familiarity with machine learning fundamentals

## Tools & Frameworks
- Python
- PyTorch
- NumPy, Matplotlib
- Torchvision, Hugging Face (when relevant)
